# XFlow

A Cross-Platform Engine for Asynchronous Reading and Knowledge Reshaping.
 
> **X** (未知/无限/扩展 | Unknown/Infinite/Extensible) + **Flow** (信息流/心流 | Information Flow/Flow State).

### Positioning
A dedicated "buffer layer" for deep readers. XFlow solves the problem of fragmented, perishable, and unsearchable content scattered across platforms like WeChat, X (Twitter), Zhihu, and Medium.

### Core Features
* **Unified Ingest**:
    * **Closed Ecosystems**: WeChat Official Accounts (Auto-snapshot to prevent 404s), Zhihu, Jianshu, etc.
    * **Social Media**: X (Twitter) thread stitching, Weibo, Xiaohongshu, etc.
    * **Tech Communities**: Medium, Substack, GitHub, V2EX, and personal technical blogs.
* **Digital Snapshot**: Automatically saves full-text HTML/Markdown to protect against 404 errors, private status changes, or content deletion.
* **AI Refinement**: Automatically extracts summaries and key insights to eliminate bookmark clutter and information overload.
* **Semantic Search**: Deep content indexing that supports full-text and semantic-based queries (search by "meaning" rather than just "keywords").
* **Extensible Ecosystem**: Supports custom annotations, cross-platform synchronization, and exports to third-party knowledge bases (Notion/Obsidian).

### Use Cases
* **Capture**: Send high-value content from mobile or PC to XFlow for asynchronous processing with one click.
* **Process**: Read in a distraction-free mode, record interactive annotations, and generate AI-driven deep summaries.
* **Retrieve**: Quickly locate specific insights across tens of thousands of records using semantic search, breaking down knowledge silos.
